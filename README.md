# Github API in Python

Technologies:
----------
- Flask
- MongoDB
- TailwindCSS
- Docker

Steps to Run:
----------
```bash
  1. Clone repository
  2. Copy .env.example to .env
  3. Run $docker-compose up -d
  4. Go to MongoExpress (http://localhost:8081) and create a database named: users
  5. Open http://localhost:5000
```
